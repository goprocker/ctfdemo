# OBSCURA 2026 | Capture The Flag

The flagship techno-cultural fest of **Sathyabama Institute of Science and Technology**. A collision of innovation, tradition, and arcane digital mastery.

## 🚀 Tech Stack

- **Core Framework**: [Next.js](https://nextjs.org) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS Context
- **Icons**: [Lucide React](https://lucide.dev/)
- **Audio Engine**: Custom React Context / Global Audio Component
- **Typography**: 
  - *Harry Potter* (Hero Section)
  - *Cinzel* (Headings)
  - *Inter* (Body)
  - *Cormorant Garamond* (Decorative)

## 🛠️ Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app/`: Next.js App Router pages and layouts.
- `src/components/`: Reusable React components (Navbar, Footer, Countdown, Flipbook, etc.).
- `public/`: Static assets including the WiCyS Logo, Harry Potter theme, and images.

## 📜 Key Features

- **Hogwarts-Themed Hero Section**: Custom animations and fonts matching the event's lore.
- **Arcane Orders Section**: Highlighting the different CTF categories (Umbra, Crypta, Vulnus, Inversus).
- **Interactive Rule Book**: A flipbook experience for participants to browse event rules.
- **Ambient Background Music**: Immersive Harry Potter theme with user-controlled audio settings.
- **Responsive Design**: Optimized for both Desktop and Mobile experiences.
